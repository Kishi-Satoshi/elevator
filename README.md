三菱電機ビルソリューションズ「AXIEZ-LINKs」の公開カタログ情報を参考にした、
かご内意匠と乗り心地を3D空間で体験できる非公式コンフィギュレーターです。
公式ロゴ・実機音声・写真素材は使用していません。

## 体験の流れ

1. **イントロ**: タイプ（P乗用 / R住宅用）と定員（6〜15名）を選択
2. **扉が開く**: ゲート演出とともにかご内へ
3. **カスタマイズ & 乗車**: 右パネルで意匠を変えながら、フロアボタンで実際に乗車

## 機能

### 乗車体験
- **操作盤(COP)は扉横の前面壁**（実機同様のレイアウト）。乗車後は扉が閉まり、正面右の大型3Dボタンがはっきり見える。初期視点も扉側を向く
- **非常ボタン & ペットボタン**（COP下段）: **非常ボタン**を押すと非常アナウンス＋ブザーが鳴り、画面が赤く明滅して「非常ボタン作動中」のインジケーターが点灯（もう一度押すと解除）。**ペットボタン**を押すとかご内にブロックのイヌ／ネコが現れてトコトコ歩き回る（鳴き声つき・最大5匹）
- 8フロアの行先ボタン（UIパネル / **かご内の大型3Dボタンも直接クリック可能**・刻印/沈み込み/発光フィードバックつき。登録階は到着まで点灯。**ダブルクリックで登録を取消**）
- **定員は 6 / 15 / 50 / 90 名（最大90人乗り）**。定員に応じてかごの幅・奥行・天井高がスケール
- **俯瞰ビューではかご外の構造**（巻上ロープ・かご枠・綱車・ガイドレール・つり合いおもり・緩衝器）も見える
- **行先キュー式（SCAN制御）**: 走行中でも別の階を登録でき、進行方向の手前に押した階へは先に停車してから元の行先へ継続（本物のエレベーターと同じ挙動）
- 走行シーケンス: 戸閉 → 加減速（S字）に追従する走行音・振動 → 階通過表示 → 到着チャイム（上り単音 / 下り二連音）→ 戸開
- **ボクセルフロア探索（一人称）**: ドアの外はブロックの世界。「フロアに出る」で主人公目線になり、WASD+マウスで売場を歩き回れる（Space ジャンプ・重力・ブロック衝突つき。**1段の段差は自動でのぼる**）
  - **本家ボクセルゲーム式の描画エンジン**: 露出面だけを統合したチャンクメッシュ + **頂点AO（角の落ち影）** + 面方向の明暗ベイク。数千ブロックでも数ドローコールで軽快に動き、ブロックの立体感が本家のルックに近づく
  - **移動フィール**: **W二連打/Ctrlでダッシュ**（FOVが広がる）・**Shiftでスニーク**（減速+端から落ちない）・歩行に同期したヘッドボブと**ブロック素材ごとの足音**
  - **採掘**: 左クリック**長押し**でゲージが進み、**5段階のひび割れ**が深くなって破壊（素材ごとに硬さが違う）。画面には**手持ちブロック**が表示され、攻撃/設置でスイングする
  - **右クリックで選択中のブロックを置く**（1〜8キー / **ホイール** / ホットバーで選択: 木の板・石・ガラス・葉・光源・レンガ・金・ウール。押しっぱなしで連続設置）
  - **拠点の保存**: 編集した売場はセッション中保持され、**B キーで localStorage に保存**（自動保存つき）。再訪してもブロックの拠点が残る。人形・敵は来店ごとに復活
  - しばらくするとエレベーターは他階へ出発し、**乗場の呼びボタン（3D）**をクロスヘアで狙って呼び戻せる。開いたドアへ歩けばかごに帰還
  - **戦闘 & HP**: 敵を左クリックで攻撃すると**反撃**される（中立の買い物客も攻撃すると敵対化）。プレイヤーは**ハートのHP**を持ち、被弾で減少、0で**ゲームオーバー→やり直し**（HP全回復してかご帰還）
  - **敵モブ（マインクラフト風）**: ゾンビ（腕を突き出して追跡）/ スケルトン（俊足）/ スライム（跳ねて接近）/ クリーパー（接近で導火線→爆発しブロックを破壊）
  - **昼夜サイクル**: **太陽が東から西へ軌道を描き**、夕暮れの茜色→**月と星空**へ遷移（ブロックの明るさも連動、画面上部に時刻HUD）。**夜になると敵モブがプレイヤー周辺に増援**として湧く。遠景はフォグで奥行きが出る
  - **地下(B1〜B3) & 溶岩**: 1Fの床を掘り抜くとデパ地下→駐車場→機械室と降りられる。**最深部B3の床下はマグマ溜まり** ─ 溶岩に触れると熱で継続ダメージ（Spaceで浮上して脱出、ブロックで橋を架けられる）
  - **アイテム収集**: たおした敵はコイン/ほね/ジェムを、葉はりんご・金鉱脈はコインをドロップ。近づくと吸い寄せられて自動収集され、右下のインベントリに加算される
  - **全フロアが開けたフィールド**: 部屋ではなく、各階の特徴を活かした開放空間（空・太陽・雲つき）。1F白大理石プラザ+金の噴水 / 2Fブティックの並木ガーデン / 3F石畳のモール / 4F中庭の書架+カフェ / 5Fレンガの中庭+かまど / 6F中央ステージの広場 / 7F夜のラウンジテラス / 8F大草原（丘・木立・池・花畑）
  - ブロックのテクスチャ・人形・敵・アイテムはすべて本プロジェクトのオリジナルドット絵（ボクセルサンドボックス風の操作系）
- **百貨店フロア演出**: 到着階ごとに空気感・照明・売場サイネージ・ブロック造形の売場が各階の特徴に合わせて変わる。**かご内は現実寄りの質感のまま** — 扉が現実とブロック世界（開けたフィールド）の境界になる
- 音声アナウンス（日本語 / English / OFF）: 既定はブラウザ合成音声（自然系ボイス優先・実車音源の解析に基づく落ち着いた話速）。
  **実録クリップ対応**: `public/voice/manifest.json` とクリップを配置すると自動でそちらを再生（下記）

### 意匠カスタマイズ
- **高解像度レンダリング**: 手続きテクスチャを最大1024pxで生成し**異方性フィルタリング**を適用。壁・ドアはヘアラインのバンプ付き金属、**床はクリアコートで磨かれた石調**（環境の写り込み）。LCD・刻印・サイネージも2倍解像度
- **スタイルプリセット**: LUXURY / NATURAL / COMFORT / MODERN
- **壁・側板**: 化粧鋼板 / プレミアムウォール / ステンレス（8色）
- **床**: 樹脂タイル / プレミアムフロア（6色）
- **ドア・三方枠・幅木**: 各色（ステンレス / ブラック / シャンパンゴールド / 木目 等）
- **天井・照明**: フラット / ダウンライト白色・電球色 / コーニス間接
- **操作盤**: ステンレスクリック / クリスタル / ブラック / タッチレス（10.1型縦LCD付き）
- **オプション**: 手すり（三方）・鏡
- 定員に応じた**かご寸法スケーリング**、**マインクラフト風ブロック体型の人物**配置（少人数〜定員・車いす／ドット絵の顔つき。乗場のブロック人形と統一）で広さ・混雑感を確認

## 実録アナウンス音声の使い方（任意）

既定では合成音声ですが、お手元の録音からクリップを作って置くと自動でそちらが再生されます。

```bash
# 録音をフレーズごとに切り出して public/voice/ に出力（要 ffmpeg / numpy / scipy）
python3 scripts/slice_voice.py 録音ファイル.mp3

# 出力された clipNN.mp3 を聴いて manifest.json の割り当て
# (up / down / close / open / arrive) を修正
```

`public/voice/*.mp3` と `manifest.json` は既定で .gitignore 済みです
（録音の権利をご確認のうえ、公開したい場合のみ明示的にコミットしてください）。


## CI / 自動マージ / 自動デプロイ

| ワークフロー | 役割 |
|---|---|
| `ci.yml` | PR と `main` への push で `npm ci` → `npm run build` を実行 |
| `auto-merge.yml` | **`auto-merge` ラベル**付きPRを、CI成功後に自動で Squash マージ。`main` へのマージ後は Pages デプロイを自動起動 |
| `deploy-pages.yml` | `main` への push / 手動 / auto-merge からの起動で GitHub Pages へデプロイ |

PR に `auto-merge` ラベルを付けるだけで、CI → マージ → 公開まで全自動で流れます。

> サブパス（`/elevator/`）配信のため、ビルド時に `VITE_BASE` を渡して Vite の
> `base` を切り替えています。ローカルの `npm run dev` は `/` のままです。
